# <img src="https://images2.imgbox.com/c7/93/sDAf3Pue_o.png" width="50"/> XIcon-Converter-Pro
### Device Info Pro is a lightweight Chrome extension that displays detailed information about your device, browser, and network directly from the extension popup. It uses a neon dark interface and provides useful diagnostics and privacy insights.

![Version](https://img.shields.io/badge/version-4.0-blue)
![Manifest](https://img.shields.io/badge/manifest-v3-orange)
![Platform](https://img.shields.io/badge/platform-Chrome-green)
![Status](https://img.shields.io/badge/status-Active-success)

---

## Features 🚀

- 🌐 Public IP address detection
- 📍 City, region, and country information
- 💻 Device type detection (Desktop / Mobile)
- 🖥 Browser and Chrome version detection
- 📺 Screen resolution information
- ⚙ CPU cores and RAM detection
- 🎮 GPU renderer detection (WebGL)
- 🔋 Battery level and charging status
- 🌍 WebRTC local IP leak detection
- 🛡 Basic VPN / Proxy detection
- 📋 Copy all information to clipboard
- 📁 Export device report as JSON
- 🌙 Neon dark UI design
---

## 🛠 Installation (Chrome Extension)

1.Download or clone this repository

2.Open Chrome

3.Go to chrome://extensions/

4.Enable Developer Mode

5.Click Load Unpacked

6.Select the project folder

Done ✔
